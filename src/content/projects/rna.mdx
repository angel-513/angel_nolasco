# Red Neuronal Aritificial convolucional

El proyecto es una red neuronal convolucional desarrollada en Python haciendo uso de las librerías de pytorch, pandas y numpy. El objetivo era desarrollar una inteligencia artificial limitada a identificar 3 clases de figuras geométricas (triángulos, cuadrados y círculos) mediante imágenes. Las entradas de datos se hacen a través de una web cam conectada al dispositivo que ejecuta el programa, desde la cual se le presenta una serie de figuras dibujadas a mano y la inteligencia artificial tiene la capacidad de categorizar dichas figuras y mostrar los resultados en pantalla.
